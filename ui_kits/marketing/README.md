# Marketing Website — UI Kit

The public-facing BIA Finance Club website. Editorial, restrained, navy + gold.

## Components
- `Nav.jsx` — sticky top navigation with wordmark
- `Hero.jsx` — homepage hero with eyebrow + display headline + lede
- `EventsList.jsx` — calendar list with date column + meta
- `Leadership.jsx` — leadership grid with portrait placeholders
- `Membership.jsx` — application call to action + criteria
- `ResearchList.jsx` — publications index
- `Contact.jsx` — contact section with hairline form
- `Footer.jsx` — wordmark, links, colophon

## Screens (`index.html`)
A scrollable single-page composition demonstrating Hero → Events → Research → Leadership → Membership → Contact → Footer.
